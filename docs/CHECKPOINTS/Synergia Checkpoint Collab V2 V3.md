COLLAB v2 es el sistema de ejecución paralela de agentes IA trabajando sobre un mismo proyecto con contexto compartido.

🧩 Características
Agentes ejecutándose en paralelo
Contexto común del proyecto
Outputs complementarios
Coordinación básica vía orchestrator
🤖 Agentes activos

🚀 COLLAB v3 — EVOLUCIÓN FUTURA
📌 Definición

COLLAB v3 es un sistema de agentes con orquestación inteligente, memoria compartida y toma de decisiones autónoma.

🧠 Características clave
Orchestrator con decisión automática
Memoria global compartida entre agentes
Loop continuo de análisis → ejecución → mejora
Auto-selección de agentes según tarea
Flujos de trabajo autónomos
🔄 Diferencia con v2
Versión	Comportamiento
v2	ejecución paralela manual
v3	sistema autónomo con decisión
