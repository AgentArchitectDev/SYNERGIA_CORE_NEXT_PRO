# 🧠 BACKEND — SYNERGIA CORE NEXT

## 📁 Ubicación
backend/

---

## 🎯 FUNCIÓN
Es el núcleo del servidor del sistema.

---

## ⚙️ RESPONSABILIDADES

1. Levantar API principal (FastAPI)
2. Recibir requests del frontend/editor
3. Coordinar IA (ai/)
4. Orquestar render (render/)
5. Manejar proyectos (projects/)
6. Controlar storage (storage/)

---

## 🔁 FLUJO

Usuario → API → backend → IA → render → frontend

---

## 🔗 CONEXIONES

- ai/ → inteligencia
- render/ → HTML final
- storage/ → persistencia
- editor/ → interfaz visual

---

## 🧠 IMPORTANCIA
Es el “cerebro operativo” del sistema en runtime.
